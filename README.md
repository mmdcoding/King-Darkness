# 👑 King Darkness PvP Resource Pack

## ⚔️ Overview
King Darkness is a premium PvP resource pack designed with a dark, clean, and competitive visual style.

It focuses on clarity, performance, and a strong black-red PvP identity.

---

## ✦ Features
- 🗡️ Custom Sword PvP textures for clear combat visibility  
- ⚒️ Refined Axe PvP textures for clean hit feedback  
- 🔨 Unique Mace PvP textures with heavy-impact design  
- 💎 Crystal PvP (CPvP) dark-themed assets  
- 🧱 Clean Carpet PvP textures for better arena visuals  
- 🎯 High-contrast black & red color theme  
- ⚡ FPS-optimized textures for smooth gameplay  

---

## ✦ Design Philosophy
This pack is built for competitive PvP players who want:

- Maximum visibility in fights  
- Minimal visual clutter  
- A strong dark-themed identity  

Every texture is designed to improve gameplay performance and combat focus.

---

## ⚙️ Performance
- Lightweight and FPS-friendly design  
- Optimized textures for PvP servers  
- Reduced unnecessary visual noise for better focus
